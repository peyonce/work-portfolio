 # My Portfolio

##  Live Demo
**Visit:** https://my-portfolio-2026-dfc71.web.app

##  Tech Stack
- React 19 + TypeScript
- Vite
- Firebase Hosting
- React Simple Typewriter

##  Features
- Fully responsive design
- Hero section with animated typewriter
- Projects showcase with role and tech stack
- Team project highlight
- Contact section
- CV download

## Deployment
```bash
npm run build
firebase deploy --only hosting

src/
├── components/     # Navbar, Footer, ProjectCard

├── styles/         # CSS files
